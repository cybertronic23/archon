# Changelog - v0.1.0

## [0.1.0] - 2026-04-08

### 🎉 初始版本发布

Archon 的初始版本，提供基础的 AI Agent 功能，包括核心工具集、LLM 集成、会话管理和权限控制。

---

### ✨ 新增功能

#### 核心架构
- **Agent Loop** - 完整的 Agent 循环，支持工具调用和响应处理
- **Tool Registry** - 可扩展的工具注册和执行系统
- **Session Management** - 会话保存/加载，支持持久化对话历史
- **Permission System** - 基于风险级别的权限控制框架
- **Context Compression** - 自动上下文压缩，防止 token 超限

#### 内置工具
| 工具 | 描述 |
|------|------|
| `read` | 读取文件内容，支持偏移量和行数限制 |
| `write` | 创建或覆盖文件，自动创建父目录 |
| `edit` | 精确的字符串替换，支持 replace_all |
| `bash` | 执行 shell 命令，支持 Docker 沙盒 |
| `glob` | 文件模式匹配，支持递归搜索 |
| `grep` | 正则搜索，自动跳过二进制文件和隐藏目录 |
| `web_fetch` ⭐ | 获取网页内容，转换为 Markdown |
| `web_search` ⭐ | 使用 DuckDuckGo 搜索网络 |

#### LLM 提供商支持
- **Anthropic** - Claude 系列模型
- **OpenAI** - GPT 系列及兼容 API
- **DashScope** - 阿里云模型
- 流式响应 (SSE)
- 自动重试机制

#### 安全与权限
- 三级风险分类：Safe / Moderate / Dangerous
- 交互式权限提示
- Docker 沙盒：Off / Permissive / Strict
- `always_allow` 缓存

#### 会话与配置
- 自动保存/恢复会话
- 历史记录 (rustyline)
- 配置文件 `~/.archon/config.toml`
- 环境变量支持

### 📝 技术细节

#### 依赖栈
- **语言**: Rust 2021 Edition
- **异步**: Tokio
- **HTTP**: reqwest
- **容器**: Docker (bollard)
- **CLI**: clap + rustyline
- **配置**: TOML

#### 项目结构
```
crates/
├── archon-core/     # 核心类型和 Agent 循环
├── archon-llm/      # LLM 提供商实现
├── archon-tools/    # 工具实现
└── archon-cli/      # 命令行界面
```

---

### 🔮 已知限制

1. **无 MCP 支持** - 无法连接外部数据源
2. **无向量存储** - 缺乏长期记忆
3. **无多 Agent 协作** - 只能单实例运行
4. **无 LSP 集成** - 缺少代码补全
5. **网络依赖** - 编译和运行需要网络
